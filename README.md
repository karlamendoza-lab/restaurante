# Restaurante Salvadoreño

Proyecto web de un restaurante típico salvadoreño donde los clientes pueden realizar reservas y conocer la gastronomía del restaurante.

## Sitio web

https://karlamendoza-lab.github.io/restaurante/

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Reserva de mesas
- Visualización del menú
- Información del restaurante
- Diseño responsive
